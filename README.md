# ResumeForge — Professional Resume Builder

A professional resume builder web app to create, customize, and download a resume as PDF.

## ✨ Features
- 📝 Personal Info, Summary, Experience, Education, Skills & Projects
- 🎨 Multiple Templates — Classic, Modern, Minimal
- 👁️ Live Preview as you type
- ⬇️ Download resume as PDF
